# JCR / Top-Journal Writing Standards for Consumer Psychology

**Target:** Journal of Consumer Research, Journal of Marketing Research, Journal of Consumer Psychology, Journal of Marketing.

---

## Manuscript Structure

### Standard Sections (APA 7th)

1. **Title Page** — title, running head, author note, word count
2. **Abstract** — ≤ 150 words; include contribution, method, key finding
3. **Introduction** — hook → gap → contribution → roadmap (≤ 2 pages for JCR)
4. **Theoretical Background** — theory → hypotheses (each ending with a formal Hypothesis statement)
5. **Study N** (repeat for each study):
   - Method (Participants, Design, Procedure, Measures)
   - Results (manipulation check → confirmatory → exploratory)
   - Discussion (1–2 paragraphs; link to next study)
6. **General Discussion** — summary → theoretical contribution → managerial implications → limitations → future directions
7. **References**
8. **Footnotes** (if any)
9. **Tables** (each on its own page)
10. **Figures** (each on its own page)

---

## Style Rules

### Language
- Active voice preferred ("We recruited..." not "Participants were recruited...")
- Present tense for general claims; past tense for study-specific events
- No contractions, no colloquialisms
- Hedge uncertainty appropriately: "suggests," "indicates," not "proves," "shows conclusively"

### APA Formatting
- Numbers: spell out one through nine; numerals for 10+; always numerals for statistics
- Statistics: `t(df) = X.XX, p = .XXX` (never `p < .05` alone; report exact p unless p < .001)
- Effect sizes: always report (d, η², ω², r, f²)
- Confidence intervals: always report 95% CIs alongside point estimates
- N per cell reported alongside overall N

### JCR-Specific Conventions
- Contribution statement in introduction: "This research contributes to X by Y"
- Each study must have a brief purpose statement at its outset
- Conceptual replications should note how they extend prior work
- Exploratory findings should be labeled as such
- Pilot studies described in footnotes or supplement, not main text (unless critical)

### Tables & Figures
- Table 1 is typically means/SDs/correlations
- All figures publication-quality: 300 dpi, no chartjunk, axes labeled with units
- Figure captions are self-contained (reader should understand without main text)

## Common Reviewer Concerns (Anticipate & Address)

| Concern | Pre-emptive Action |
|---------|-------------------|
| "Small N per cell" | Power analysis in Method; meta-analytic effect size as basis |
| "WEIRD sample" | Note limitations; add MTurk/Prolific diversity data if available |
| "No process evidence" | Add mediation study or measured mediator |
| "Effect driven by one item" | Report item-level analysis in supplement |
| "Demand characteristics" | Report suspicion check results |
| "Lack of external validity" | Add field study or realistic stimulus |

## Hypothesis Formatting

```
H1: [Participants in Condition A] will [show higher/lower DV] than [participants in Condition B].

H2 (moderation): The effect of [IV] on [DV] will be stronger/weaker when [Moderator is high vs. low].

H3 (mediation): The effect of [IV] on [DV] will be mediated by [Mediator].
```

Each hypothesis must appear exactly once in Theoretical Background and be tested exactly once in Results (confirmatory section).
